# Visitation-Book Style Family Tree

Compact pedigree layout inspired by heraldic visitation books (e.g. *The Visitation of the County Palatine of Lancaster*, 1613). Same orthogonal geometry as the minimal style, but with tighter spacing, smaller rectangular boxes, and a **double-line marriage connector**.

## Key points

- Smaller boxes and tighter vertical spacing than the minimal style, to fit more generations in a constrained area.
- Each marriage line is two parallel `shape=line;direction=east` segments, 6–8 SVG units apart.
- All boxes are `rounded=0` with a light neutral fill.
- The same orthogonal connector grammar applies: marriage line → vertical descender → horizontal child connector → vertical child lines.

## Export caveat

Some draw.io PNG export pipelines collapse closely-spaced parallel marriage lines into a single line. The editable diagram and SVG export will show the double lines correctly. If the PNG preview must emphasise the double line, increase the spacing between the two segments or use a heavier `strokeWidth`.
